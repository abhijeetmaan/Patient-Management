# 🏥 Smart Patient Management System (Healthcare SaaS)

A full-stack healthcare management platform designed for doctors to manage patients, appointments, and insights efficiently.

## 🚀 Features

- 🔐 JWT Authentication (secure login system)
- 👨‍⚕️ Multi-doctor support (data isolation)
- 🧑‍💼 Admin panel (manage doctors & patients)
- 📊 Analytics dashboard (charts & insights)
- 📅 Appointment scheduling + calendar view
- 📌 Appointment status tracking
- 🔔 Notification system
- 📄 Prescription generator
- 💳 SaaS billing system (Free / Pro / Premium)
- 🧠 Smart insights from patient data

## 🛠 Tech Stack

- Frontend: React.js + Tailwind CSS
- Backend: Node.js + Express
- Charts: Recharts
- Auth: JWT
